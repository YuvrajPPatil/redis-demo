# Redis Cache API (Node.js)

## Overview
This project demonstrates API caching using Redis to improve performance.

## Flow
Client → Node API → Redis (cache)
                 → External API (if cache miss)
## Tech Stack
- Node.js
- Express
- Redis
- Axios

## Endpoint
GET /todo

## How to run
- start docker
- start redis
- run project
